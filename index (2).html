Case Study 01 · AI Chatbot

Owlex — Building a context-aware conversational AI from scratch

Problem

Generic chatbots lose context between turns, giving irrelevant responses and high user drop-off.

Solution

Built with RAG — vector DB stores knowledge, LLM retrieves precise context before each response. Flask API for easy integration.

↑ UX
Result

Context-aware conversations, scalable architecture, production-ready API endpoints.

Case Study 02 · NLP + Speech AI

Foul Language Detection — Real-time multilingual moderation at scale

Problem

Platforms had no real-time tool for detecting offensive audio in both Hindi and English — a massive gap in the Indian market.

Solution

End-to-end pipeline: Google Speech-to-Text → custom NLP classifier → Flask REST API → MySQL tracker. Fully modular and reusable.

92%
Result

92% accuracy across both languages. Published research paper. Architecture reusable on any audio platform.
